## 개발환경 구축
- Front-End : React 18.2, 3000 Port
- Back-End  : Servlet(Java-JDK21, Tomcat 11, 8000 Port)

### Front-End
#### 설치한 외부 라이브러리
```sh
# 비동기통신을 위한 라이브러리
npm i axios

# 화면 처리 관련 라우터 라이브러리
npm i react-router

# 반응형과 React 버전도 지원되는 UI 라이브러리(CSS, JS)
npm i bootstrap

# 클라우드 API를 활용한 로그인, NOSQL 제품 활용, 파일 업로드
npm i firebase

# React 라이브러리, React DOM 라이브러리
npm i react
npm i react-dom
```

#### 프로젝트 디렉토리
- src
  - assets : 이미지소스
  - components 
    - auth  : 인증, 인가, 회원가입, ...
    - depth : 부서목록, 부서수정, 부서삭제, 부서상세
    - include : Header, Footer
    - pages : 
      - 부서관리(DeptPage)
      - 사원관리(EmpPage)
      - 홈페이지(HomePage)
  --- [[모델 계층 - 비즈니스로직 + 데이터셋]] ---
  - service
    - firebase
    - authLogic
    - depthLogis
    - empLogic
  - redux(Flux Architecture)
    - 상태관리
    - redux toolkit(외부 라이브러리)

#### 실헹 흐름
```sh
# 서버 기동하기
npm start
```
1. src > index.js 호출
2. public > index.html에서 root 찾기
3. Router 설정 추가 -> index.js - APP compenent
4. 페이지 전환 가능 - endpoint
  - http://localhost:3000       : HomePage.jsx
  - http://localhost:3000/login : LoginView.jsx
  - http://localhost:3000/dept  : DeptPage.jsx
  - http://localhost:3000/emp   : EmpPage.jsx
5. index.js에서 App.js
```js
  <Route path = '/'      exact = {true} element = {<Homepage />} />
  <Route path = '/login' exact = {true} element = {<LoginForm />} />
  <Route path = '/dept'  exact = {true} element = {<DeptPage />} />
```
